Project Overview

Developed a Content-Based Image Retrieval system using MATLAB and machine learning to retrieve images from a large database based on visual similarity. The system achieved 81% accuracy in image retrieval and was published in the International Research Journal of Engineering and Technology (IRJET).

Objectives

  • Build an efficient image retrieval system without relying on text metadata
  • Extract visual features from images using computer vision techniques
  • Implement machine learning algorithms for image matching
  • Achieve high accuracy in retrieving relevant images
  • Create a scalable system for large image databases
  • Publish research findings in academic journals

Methodology

I implemented a comprehensive CBIR pipeline using the Bag-of-Features approach with local image descriptors and machine learning classification:

Key Achievement: Achieved 81% retrieval accuracy on 10,000+ images using Bag-of-Features model with SIFT descriptors and SVM classifier.

Technical Approach:

  • Feature Extraction: Used SIFT (Scale-Invariant Feature Transform) to detect local features
  • Feature Representation: Created visual vocabulary using k-means clustering (Bag-of-Features)
  • Image Representation: Represented each image as histogram of visual words
  • Classification: Used Support Vector Machine (SVM) for image matching
  • Similarity Computation: Calculated chi-square distance for image ranking
  • Performance Optimization: Implemented indexing for fast retrieval

Technical Specifications

  • Vocabulary Size: 500 visual words (k-means clusters)
  • Classifier: One-vs-All SVM with RBF kernel
  • Distance Metric: Chi-square distance for histogram comparison
  • Database: 10,000+ images across 10 categories
  • Retrieval Time: Average < 1 second per query
